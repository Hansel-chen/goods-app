# 倒货管理小程序使用说明

## 📱 项目简介

这是一个用于管理倒货信息的微信小程序，实现了商品的增删改查功能。数据存储在微信本地存储中，无需后端服务器。

## 🚀 如何上传到微信小程序

### 1. 准备工作

#### 下载微信开发者工具
- 访问：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- 下载并安装适合你系统的版本

#### 注册微信小程序账号
- 访问：https://mp.weixin.qq.com/
- 点击"立即注册" → 选择"小程序"
- 填写相关信息完成注册
- 登录后在"开发" → "开发管理" → "开发设置"中获取 **AppID**

### 2. 导入项目

1. 打开微信开发者工具
2. 点击"+"号或"导入项目"
3. 填写项目信息：
   - **项目目录**：选择 `C:\Users\HanZhe.Chen\倒货小程序`
   - **AppID**：填入你在微信公众平台获取的AppID（如果只是测试，可以选择"测试号"）
   - **项目名称**：倒货管理系统

### 3. 本地调试

1. 导入成功后，开发者工具会自动打开项目
2. 在模拟器中可以看到小程序的界面
3. 可以进行增删改查操作测试

### 4. 上传代码

1. 点击开发者工具右上角的"上传"按钮
2. 填写版本号（如：1.0.0）和项目备注
3. 点击"上传"

### 5. 提交审核

1. 登录微信公众平台：https://mp.weixin.qq.com/
2. 进入"管理" → "版本管理"
3. 在"开发版本"中找到刚上传的版本
4. 点击"提交审核"
5. 填写相关信息并提交

### 6. 发布小程序

1. 审核通过后，在"审核版本"中可以看到审核通过的版本
2. 点击"发布"即可正式发布小程序

## 📋 功能说明

### 1. 列表页面（首页）
- **查看所有商品**：显示所有已添加的商品信息
- **搜索功能**：可以根据商品名称或编号搜索
- **统计信息**：显示总数量和总金额
- **快速操作**：每条记录可以直接编辑或删除

### 2. 添加页面
- 点击"添加新商品"按钮进入
- 填写商品信息：
  - **必填项**：商品名称、商品编号、数量、单价、日期
  - **选填项**：状态、供应商、联系电话、备注
- 点击"提交"保存数据

### 3. 编辑页面
- 在列表页点击"编辑"按钮进入
- 修改商品信息
- 点击"保存"更新数据

### 4. 详情页面
- 点击列表中的商品卡片进入
- 查看完整的商品信息
- 可以进行编辑或删除操作

## 📝 数据字段说明

当前小程序包含以下数据字段，你可以根据实际需要调整：

- **商品名称**（name）：商品的名称
- **商品编号**（code）：商品的唯一编号
- **数量**（quantity）：商品数量
- **单价**（price）：商品单价
- **日期**（date）：交易或记录日期
- **状态**（status）：正常/停用
- **供应商**（supplier）：供应商名称
- **联系电话**（phone）：联系电话
- **备注**（remark）：其他备注信息

## 🔧 自定义修改

### 如何添加新字段

如果你需要添加新的数据字段，需要修改以下文件：

1. **添加页面**：`pages/add/add.wxml` 和 `pages/add/add.wxss`
2. **编辑页面**：`pages/edit/edit.wxml` 和 `pages/edit/edit.wxss`
3. **详情页面**：`pages/detail/detail.wxml`
4. **列表页面**：`pages/list/list.wxml`（如果需要在列表中显示）

示例：添加一个"产地"字段

在 `pages/add/add.wxml` 中添加：
```xml
<view class="form-item">
  <view class="label">产地</view>
  <input class="input" name="origin" placeholder="请输入产地" value="{{formData.origin}}" />
</view>
```

### 如何修改字段名称

直接在对应的 `.wxml` 文件中修改显示的文字即可。

## ⚠️ 注意事项

1. **数据存储**：数据存储在微信本地存储中，卸载小程序或清除缓存会导致数据丢失
2. **数据限制**：微信本地存储有 10MB 的限制
3. **测试建议**：先使用测试号充分测试后再申请正式的小程序账号
4. **审核要求**：提交审核时需要提供小程序的使用说明和隐私政策

## 🆘 常见问题

**Q: 如何备份数据？**
A: 目前数据存储在本地，建议定期导出重要数据。后续可以考虑接入云存储。

**Q: 可以多人使用吗？**
A: 每个用户的数据是独立的，存储在各自的微信客户端中。

**Q: 如何清空所有数据？**
A: 可以在小程序中添加一个"清空数据"功能，或者直接删除小程序重新打开。
